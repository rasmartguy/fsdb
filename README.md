# fsdb


fsdb is one of Rasmart node's module.Presenet oneself as KeyValue small database

### How-to
```sh
    cd net
    mkdir build && cd build
    cmake .. 
    cmake --build .
```

### Supported compilers
    - gcc 4.8+
    - clang 3.3+
    - MCVS 15/17
